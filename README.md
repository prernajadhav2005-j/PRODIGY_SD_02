1. Project Overview : The Guessing Game Android App is a simple and interactive mobile application developed using Java and XML. The app generates a random number and
    challenges the user to guess it. Based on the userâ€™s input, the application provides instant feedback indicating whether the guess is too high or too low. The game
    continues until the correct number is guessed.This project is designed to demonstrate basic Android development concepts, including user input handling, conditional
    logic, random number generation, and UI design.

3. Features :
- Random number generation
- Feedback for high and low guesses
- Attempts counter display
- Reset game functionality
- Clean and user-friendly UI
- Splash screen on app launch

3. Technologies Used :
  - Frontend : UI Design by using XML
  - Backend : Java
  - IDE: Android Studio
  - Android SDK

4. How the App Works :
 - The app starts with a splash screen.
 - A random number between 1 and 100 is generated.
 - The user enters a guess.
 - The app provides feedback:  Too High
                               Too Low
 - The attempt count increases after each guess.
 - The game ends when the correct number is guessed.
 - The user can reset the game to play again.
